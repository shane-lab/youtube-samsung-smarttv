### YouTube HTML5 Samsung SmartHub TV app

A private, re-constructed Angular application for the the deprecated [**YouTube Flash**](https://support.google.com/youtube/answer/7379403?hl=en) app for _Samsung SmartHub TVs (2012 - 2014)_.

#### notice
Currently not open for public use. I have yet to find a way to sign in using the Google OAuth sigin from the Samsung Smart TV interface.
